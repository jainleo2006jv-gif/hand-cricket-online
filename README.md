# 🏏 Hand Cricket Online

A premium single-file Hand Cricket website built with **HTML, CSS, and JavaScript**.

## ✨ Features
- Beautiful premium dark UI
- Home page with animated design
- User login / register (localStorage demo auth)
- Play vs CPU
- 2 Player mode
- Online room simulation
- Leaderboard / scoreboard
- Profile page
- Sound effects
- Confetti win effects
- Mobile responsive design

## ⚠️ Important Note
This project currently uses:
- **localStorage for auth**
- **localStorage for scoreboard**
- **localStorage-based online room simulation**

So this is best for:
- College projects
- Portfolio showcase
- Demo website
- GitHub Pages hosting

It is **not a real backend multiplayer app** yet.

## 🚀 Live Demo (GitHub Pages)
After uploading to GitHub, enable **GitHub Pages**:
- Go to **Settings**
- Open **Pages**
- Under **Build and deployment**
- Select **Deploy from a branch**
- Choose:
  - Branch: `main`
  - Folder: `/ (root)`
- Save

Your site will be live at:
